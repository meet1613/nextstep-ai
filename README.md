
# NextStep AI

NextStep AI is a simple AI productivity assistant that suggests the next best action to achieve a goal.

## Features
- Enter a goal
- Enter available time
- AI suggests the next best step

## Tech Stack
- Python
- Flask
- HTML
- CSS
- JavaScript
- OpenRouter API

## How to Run

1. Clone the repository
2. Install dependencies
3. Add your API key in .env
4. Run:

python3 app.py

Then open:
http://127.0.0.1:5050
